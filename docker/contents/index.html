Welcome to BioTerms.org

BioTerms.org is a community initiative to define and promote schemas for structured epigenome annotation data. These schemas enable researchers to share epigenome datasets through a common vocabulary and structure.

BioTerms schemas are built in JSON-LD (JavaScript Object Notation for Linked Data) and are based on generic terms from schema.org. It is also heavily based on the Sequence Ontology for epigenome-specific entities and relationships. BioTerms thereby effectively links biological ontologies to generic schemas.

For example, here is a portion of the schema defining the start of a genome segment:

    {
      "@id": "http://bioterms.org/schemas/segmentStart",
      "@type": "rdf:Property",
      "https://schema.org/domainIncludes": {
        "@id": "http://bioterms.org/schemas/Segment"
      },
      "https://schema.org/rangeIncludes": [
        {
          "@id": "https://schema.org/Number"
        },
        {
          "@id": "https://schema.org/Text"
        }
      ],
      "rdfs:comment": "Start of a segment",
      "rdfs:label": "segmentStart"
    }
